Q1. 我什麼時候會收到激燃體？

更新於 2020 / 03 / 20 16 : 11

Q2. 我要怎麼收到激燃體呢？

更新於 2020 / 03 / 20 16 : 11
合作夥伴 justfont 會發佈一封啟用信件到您在嘖嘖登記的信箱，啟用之後就會導引到 justfont.com 進行下載囉！（所以要記得正確的信箱喔～

Q3. 我是 Mac 電腦，可以使用激燃體嗎？

更新於 2020 / 03 / 20 16 : 11
可以！Windows, Linux 等 OS 也都可以喔

Q4. 請問可以使用在 iWork、Office、Final cut、Adobe Premiere/Illustrator/Photoshop、Inkscape、威力導演等軟體中嗎？

更新於 2020 / 03 / 20 16 : 11
沒問題喔～各大正版最新設計、文書軟體皆支援。若您屬於較為舊版的軟體，可以去信 just@justfont.com 客服信箱，索取 ttf 檔案，安裝會更為順利喔～

Q5. 有辦法使用在 iPhone、iPad Pro 等行動裝置中嗎？

更新於 2020 / 03 / 20 16 : 11
根據 justfont 最新的通用授權條款，您可以將激燃體使用於 iPad 等平板電腦上作為編輯、設計（例如用於 Procreate 中）使用喔～

Q8. 我是獨立創作／接案者，可以將激燃體用在客戶的商業案件上嗎？

更新於 2020 / 03 / 20 16 : 11
可以的～您可以使用在委託您設計的案件中；但如果是客戶自己指定用字，建議還是由客戶購買字型的授權費用比較合理喔！

Q13. 以後會出 web font 網頁字型嗎？

更新於 2020 / 03 / 20 16 : 11
隨著激燃體正式推出之後，就會將 Web Font 版本上架喔。正體預計於 2021 年 1 月後實現，斜體預計於 2021 年 5 月後實現。

Q14. 請問你們如何防止盜版？

更新於 2020 / 03 / 20 16 : 11
您日後將使用的激燃體字型檔都是獨特的。若有盜版之情況發生，我們可依據字型檔案內容查詢來源並追究之。但不論有多好的機制，都不能完全防止盜版。盜版是傷害創作者的行為，尤其創作者非常認真的在製作每一個字，要享受好的創作內容，我們誠摯希望每一個人都該從尊重版權做起。

Q16. justfont vvip 是什麼呀？

更新於 2020 / 03 / 20 16 : 11
若您是 justfont vvip，代表您曾經參與 justfont 重要的里程碑，支持新創字型產業發展，是 justfont 非常、非常重要的貴人 (very very important person, vvip)。舉例而言，2015 年的金萱、2019 的凝書體贊助者、2018 參與金萱那提字型早鳥預購的消費者，都自動擁有此資格。
